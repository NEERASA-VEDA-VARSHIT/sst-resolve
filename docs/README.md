# SST Resolve

A ticketing platform built with Next.js 15, Clerk, Drizzle ORM, and shadcn/ui.

## Getting Started

1. Install deps: `pnpm install`
2. Set env vars in `.env.local` (see `drizzle.config.ts`, Clerk, Slack, SMTP)
3. Run DB migrations: `npx drizzle-kit push`
4. Start dev server: `pnpm dev`
